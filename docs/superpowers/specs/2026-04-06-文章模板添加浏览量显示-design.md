# 文章模板添加浏览量显示 - 设计文档

## 项目概述
为Hexo博客的NexT主题添加文章浏览量统计显示功能，使用Busuanzi（不蒜子）统计服务。Task 1已完成Busuanzi配置，Task 2需要修改文章模板来显示浏览量。

## 需求
- 在每篇文章末尾显示浏览量统计
- 只在文章详情页（post页面）显示，不在文章列表页（index页面）显示
- 使用主题原生Busuanzi支持
- 显示位置：文章标签之后、分享按钮之前

## 设计方案

### 位置选择
文件：`themes/next/layout/_macro/post.njk:120-130`
具体位置：标签代码块（第120-127行）和分享按钮（第129行）之间

### 代码实现
```njk
{# 文章浏览量显示 - 放在标签之后、分享按钮之前 #}
{%- if not is_index and theme.busuanzi_count.enable and theme.busuanzi_count.post_views %}
  <span class="post-meta-item" title="{{ __('post.views') }}" id="busuanzi_container_page_pv">
    <span class="post-meta-item-icon">
      <i class="{{ theme.busuanzi_count.post_views_icon }}"></i>
    </span>
    <span class="post-meta-item-text">{{ __('post.views') + __('symbol.colon') }}</span>
    <span id="busuanzi_value_page_pv"></span>
  </span>
{%- endif %}
```

### 条件判断逻辑
1. `not is_index` - 只在文章详情页显示
2. `theme.busuanzi_count.enable` - Busuanzi功能已启用
3. `theme.busuanzi_count.post_views` - 文章浏览量显示已启用

### 样式说明
- 使用`post-meta-item` CSS类，与主题其他元数据元素保持一致
- 图标使用`theme.busuanzi_count.post_views_icon`配置
- 文本使用主题翻译函数：`__('post.views')`和`__('symbol.colon')`
